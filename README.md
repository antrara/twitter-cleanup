# Twitter Cleanup 🐦

Skrip ini memungkinkan Anda untuk mengelola akun Twitter secara otomatis, termasuk:
✅ Unfollow pengguna  
✅ Unlike semua tweet  
✅ Unretweet semua repost  
✅ Menghapus komentar  
✅ Menghapus tweet berdasarkan tanggal  
✅ Logout dan menghapus API Key
